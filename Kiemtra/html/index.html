What is Ajax?

As defined in Wikipedia

AJAX is a group of interrelated web development techniques used to create interactive web applications or rich Internet applications. With Ajax, web applications can retrieve data from the server asynchronously in the background withouth interfering with the display and behavior of the existing page.


Brief History

Year Event
1995 Java applets were introduced - the first way to load data from client side asynchronously
1996 IFrame element were introduced, which alse have this functionality
1999 Mircosoft create the XMLHTTP ActiveX control in Internet Explorer 5 which then followed by other browser as XMLHttpRequest object
2005 The term "Ajax" was borned
2006 The W3C, in order to do make it offical web standard , publish the first draft specification for XMLHttpRequest Object
